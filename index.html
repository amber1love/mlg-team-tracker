📋 How To Use
Quick reference for the MLG Team Tracker.

Tabs Overview

  • Deal Tracker — Master list. All other tabs pull from here.
  • Quarterly Forecast — Deals grouped by close month into Q1–Q4. Rep scorecards show pipeline vs 3x quota.
  • Weekly Notes — Per-deal coaching notes by week. MEDDPICC inline. History dropdown for past weeks.
  • Weekly Forecast — Thursday-by-Thursday forecast. Commit entered manually. Closed Won and Stage 6/7 auto-pull.

Status Triggers

  • 🟢 Active — Engaged, next steps agreed
  • 🟡 Stalled — No activity 14+ days
  • 🔴 Not Live — Lost, on-hold, or dark

Forecast Buckets

  • Commit — High confidence, committing to close
  • Best Case — Could close with upside scenario
  • Pipeline — In funnel, not yet qualified
  • Upside — Stretch / opportunistic

MEDDPICC

  • M — Metrics: quantified ROI
  • E — Economic Buyer: budget authority
  • D — Decision Criteria / Process / Date
  • P — Paper Process: legal/procurement
  • I — Identify Pain: compelling event
  • C — Champion + Competition

Weekly Forecast Columns

  • Commit — Manually entered each Thursday
  • Variance — Change from prior week commit
  • Closed Won — From Deal Tracker (read-only)
  • Stage 6/7 — Finance Hold/Review (read-only)
  • Pipeline columns — CW + buckets cumulative
