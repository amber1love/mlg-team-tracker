📋 How To Use This Forecasting Workbook
A complete guide to sheets, auto-routing, status triggers, and MEDDPICC methodology.

Sheet Overview

  • Deal Tracker — Master pipeline. Add every deal here. Status dropdown triggers color-coding (🟢 Active, 🟡 Stalled, 🔴 Not Live). Weighted Value = ACV × Probability.
  • Q1/Q2/Q3 Forecast — Calculated from Deal Tracker based on Close Date. Deals flow into the correct month column automatically.
  • Weekly Notes — Tracks all active deals. Add weekly notes, progression, blockers, and action items.
  • Weekly Summary — KPI dashboard: active deals, progressed, stalled, pipeline total. Update each Monday.
  • MEDDPICC Deep Dive — Enterprise qualification table. Completeness score auto-calculates (% of 9 fields filled).

How Auto-Routing Works

  • Quarter Routing — Close Date month determines quarter: Jan–Mar → Q1, Apr–Jun → Q2, Jul–Sep → Q3, Oct–Dec → Q4.
  • Monthly Routing — Within each quarterly view, deal ACV appears in the correct month column based on Close Date.
  • Weekly Notes Flow — Weekly Notes pulls Deal Name, Account, Owner, and Status from Deal Tracker automatically.

Status Triggers

  • 🟢 Active — Engaged within 14 days, stakeholders active, next steps agreed.
  • 🟡 Stalled — No meaningful activity in 14+ days. Champion gone quiet.
  • 🔴 Not Live — Deal lost, on-hold indefinitely, or contact completely dark.
